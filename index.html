Para oque serve cada comando do github?

Para iniciantes essa pode ser uma pergunta recorrente, por isso eu vim aqui ensinar para oque cada um dos principais comandos serve

Os principais comandos são:

  • <H>

    • O comando <h> em HTML é usado para criar tiulos e subtitulos. Existem seis níveis de cabeçalhos, de <h1> a <h6>, sendo <h1> o mais importante e <h6> o menos importante. Eles são usados para definir títulos e subtítulos em uma página web. Exemplo: <H> texto </H>.

  • <p>

    • O comando <p> em HTML é usado para criar parágrafos de texto. Ele marca um blochttps://github.com/AllisonMarks/Site-2024o de texto que é separado de outros blocos por espaços em branco vertical. exemplo <p> paragrafo </p>

  • <img>

    • O comando <img> em HTML é usado para inserir imagens em uma página web. Ele não tem uma tag de fechamento, e sim utiliza atributos para especificar a localização e outras características da imagem a ser exibida.

  • CSS & style

    • O CSS é usado para estilizar e controlar o design visual de páginas web, determinando cores, layout, fontes e outros aspectos visuais dos elementos HTML.

    O atributo style em HTML é usado para aplicar estilos diretamente a um elemento individualmente, sem a necessidade de um arquivo CSS separado. Ele permite definir propriedades de estilo como cor de fundo, cor do texto, margens, entre outros, diretamente dentro da tag HTML.

  • id & Class

    • id: É um atributo único que identifica um elemento HTML. Cada id em uma página deve ser único. É frequentemente usado para aplicar estilos ou referenciar o elemento em scripts JavaScript.

    class: É um atributo que pode ser aplicado a vários elementos HTML. Permite agrupar elementos que compartilham estilos semelhantes. É amplamente utilizado para aplicar estilos CSS a grupos de elementos específicos.
